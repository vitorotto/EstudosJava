# Processamento Ecomerce
Projeto simples criado para aprender java e processamento de retornos de requisições em .json\
Simple project created to learn Java and returns processing of requests in .json

# Pacotes deste projeto
Packages of this project

## Model
Neste pacote que serão descritas todas as entidades de um banco de dados\
In this package that will be describes all entities of a database

Nesta aplicação de processamento, serão descritas as entidades que vem do arquivo "loja_dados.json"\
In this application of processing, will be describe the entities that come from "loja_dados.json" file

## Repository
Este é o pacote responsável por trazer os dados do banco de dados da aplicação, ele lida diretamente com a persistência dos dados\
This is the package responsible for bringing the data from databases of application, tt deals directly with data persistence
